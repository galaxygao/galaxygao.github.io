LISTS

there are three kinds of list notations in html

< ul >

create an unordered list (with a fill dot) (using <li>to list items)
  • this is the effect of ul

< ol >

create an ordered list (with number) (using <li>to list items)
  1. this is the effect of ol

< dl >

create a descriptional list (nothing in front) (using < dd > or < dt > to list items)

< dt >

this is the effect of < dt > . there are no empty space and dt is used for title

title2

< dd >

this is the effect of < dd > . there is an indent and dd is used for contents
