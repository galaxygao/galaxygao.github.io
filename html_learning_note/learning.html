HYPERLINK

BASIC

the tag of hyperlink is <a>
the most important attribute is href, which contain the link's destination
for example i can create a link to my blog using

      
    <a href="https://galaxygao.github.io/">G_Star's Blog </a>
where href is the URL and "G_Star's Blog" is the text contents.

Effect:

G_Star's Blog

THE TARGET ATTRIBUTE

By default, the linked page will be displayed in the current browser window.
To change this, you must specify another target for the link.
The basic syntax of "target" is 

      
         <a href="URL" target="value">TEXT_NAME </a>
There are four "target" value:

_self

- Default. Opens the document in the same window/tab as it was clicked G_Star's Blog

_blank

- Opens the document in a new window or tab G_Star's Blog

_parent

- Opens the document in the parent frame G_Star's Blog

_top

- Opens the document in the full body of the window G_Star's Blog
